Crystal Collector

How to play the game

  1. The computer will choose a random number at the start of the game (aka Number to Get:)
  2. Using the Crystals
    • Blue Crystal
    • Red Crystal
    • Yellow Crystal
    • Purple Crystal
  3. You will attempt to match the "Number to Get" without going over
  4. If you match the "Number to Get", you win the game
  5. If you go over the "Number to Get", you lose the game
  6. The value of each crystal will not be determined until you click on it
  7. The game restarts whenever the player ("you") win or loss
  8. Are you ready to play? Let's GO!
